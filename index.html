HTML Content Reference

Content Tags

Tag Name Description
<h1> - <h6> Header Element represents a container for introductory content or a set of nav links.
<p> Paragraph A paragraph always starts on a new line, and is usually a block of text.
<img> Image Element that accepts the importation of external/internal images into the website. Holds many attributes listed on data below!
<video> Video With this Element we are able to accept external/internal videos in many formats and also included with many attributes!
<audio> Audio Element that lets us be able to deploy playable audio into the website with many attributes for more control!

Content Attributes

Attribute Name Description
<audio controls> Controls Audio/Video tags include attribute named "controls". Gives the ability to go back to start or end, play/pause and control volume.
<img src> Source Image/Audio/Video tags include attribute named "src" short for "Source". Gives the ability to link external/internal file with media functionality.

Code Accessibility Tags

Tag Name Description
<div> Division Tag to specify a "Division" or "Section" of a code for easier access to different parts of code!
<header> Header Easier to read and sections a head section of your code for "header" and banners" and "navs".
<aside> Aside Works as a side information for need-to-know or helpful information to explain your websites different content/sections.
<main> Main Tag used to section the "Main" content of your webpage to make the code more easily readably for other people!
<footer> Footer Defines its own section for footer information such as contacts, addresses etc. Helps alot to make code easier to read!
